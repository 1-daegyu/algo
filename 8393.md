# 문제 제목: 
n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

## 문제 정보
- **출처:** 백준문제 8393번
- **난이도:** Easy,

## 문제 설명
n을 입력 받고 1부터 n까지 합을 구함

## 해결 과정
sum = 0 으로 sum 변수를 초기화 후 합계를 저장함
반복문을 사용하여 1부터 n까지 모든 정수를 반복함
반복이 끝나면 sum에 1부터n까지 합이 저장됨
sum을 출력함

### 접근 방법
위와 비슷함 3을 입력시 1+2+3 = 6 이 출력됨

## 코드
n = input(int())
sum = 0
for i in range(1, n+1)
sum += i
print(sum)
